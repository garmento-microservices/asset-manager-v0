# Asset Manager v0

The simplest asset manager system for demonstration purposes. Not recommended for production use. Migrated from the [original monorepo](https://github.com/binhdoitsme/garmento/tree/main/infrastructure/prod).

## Running locally

```(shell)
pip install pdm
pdm install
pdm run dev
```

## Building to image

```(shell)
./build.sh
```
